Ember Promise Proxy
==============================================================================

Docorator for promise proxy array and promise proxy object.


Compatibility
------------------------------------------------------------------------------

* Ember.js v3.12 or above
* Ember CLI v2.13 or above
* Node.js v10 or above


Installation
------------------------------------------------------------------------------

```
ember install ember-promise-proxy
```
